To use IBM's Quantum computer

- Create account https://quantum.ibm.com/
- Copy API Token and set IBMQ_API_TOKEN=XXXX in .env file
- Free Up to 10 minutes/month

Python packages
`pip install -r requirements.txt`

Save credentials
`python save_credentials.py`
saved to ~/.qiskit/qiskit-ibm.json

Test connection
`python hello_world.py`
